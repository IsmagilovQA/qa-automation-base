# Automation project

##### Description

This project contains a few automation base projects which were written by Kotlin and JS.

Projects were written by **Kotlin** contains:
  - **Allure**
  - **AssertJ**
  - **Selenide**

Project was written by **JS** contains:
  - **Allure**
  - **Mocha**
  - **Chai**
